# SalaDeJuegos

Proyecto desarrollado con Angular que reúne distintos minijuegos en una misma aplicación.

## Características

- Sistema de navegación entre juegos
- Registro e identificación de jugadores
- Persistencia de puntajes
- Diferentes modos de juego
- Componentes reutilizables
- Manejo básico de estados y sesiones

## Tecnologías

- Angular
- TypeScript
- Supabase
- RxJS
- HTML / CSS
